The +81 Bar Makati website boasts an ambitious, visually striking design concept that beautifully pays tribute to high-end Japanese street fashion magazines. However, several mechanical and layout-level overlapping issues prevent it from reaching that premium "magazine cover" benchmark. Key visual assets currently clash and obscure each other, while the animations use heavy, un-decomposed transform strings that can fail to render in modern browsers. Transitioning section reveals away from template-like linear animations to focus-blur entries will elevate the visual hierarchy and match the bar's luxurious speakeasy brand.

Issues to Address

🔴 Critical

Monolithic Transform Overrides Floating Frames

The video frames (main frame and polaroids) use the monolithic CSS transform property to apply both rotation and translation loops in a single animation (transform: var(--float-base) translateY(0px)). Because keyframed transforms completely override inline transforms, browsers struggle to interpolate these dynamic variables smoothly. Additionally, on mobile sizes, the animation is simply killed (animation: none), which stops them from floating entirely. Decomposing these transforms using separate modern, independent rotate/translate properties ensures perfect responsive execution.

🟡 Important

Low Read-Contrast in Section Titles

The red and white highlight contrast (.highlight-red) in the spreads (Entrance, Craft, Rooms, Midnight) is beautiful but lacks text-shadow support. On textured backgrounds with SVG rough-filters, the text can read as low-contrast and muddy. Applying a deep editorial drop shadow similar to the hero heading will make the editorial text pop cleanly off the dark canvas.

Uniform Template-Like Slide Reveals

All spreads mount on scroll using the exact same fade-up translate transition. This uniform reveal matches standard AI templates rather than a custom-tailored Japanese street magazine.
