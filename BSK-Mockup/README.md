# BSK Club Manila Mockup

Standalone website mockup for BSK Club Manila.

## Local Preview

```powershell
python -m http.server 5192 --bind 127.0.0.1
```

Open `http://127.0.0.1:5192/`.

## Notes

- Uses local BSK photos and video from `D:\New AI Directory\BSK - Mandaluyong`.
- Public listing details used in copy: Sheridan, Greenfield District, Mandaluyong City; reservations `0917 853 4464`; Thursday to Sunday nights.
- The reservation form is a front-end mockup and can later be wired to Messenger, Instagram DM, WhatsApp, or a CRM.
